Mail Client Type

What type of mail client do you use? If you are using a local application, which is registered to handle mailto links, a new tab will quickly appear and disappear again to start this application. If you are using a web mail client the tab will be kept open for the web mail client.

Message Body Template

By default the generated Mail will only contain the page title as subject and the url to send as text. But if you want you may define text that will appear before and after the url.

Troubleshoot

If you experience problems opening mails make sure you have an application registered to handle mailto links. If such an application is registered it should allow you to write a mail by clicking this mailto link
